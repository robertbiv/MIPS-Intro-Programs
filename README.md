# MIPS-Intro-Programs

A collection of introductory MIPS assembly programs demonstrating basic programming concepts.

## Programs

- **bmi.asm** - BMI (Body Mass Index) calculator
- **caffeine.asm** - Calculate maximum drinks to reach lethal caffeine dose
- **loan.asm** - Simple interest calculator for loans
- **sqrt.asm** - Integer square root calculator
- **time.asm** - Convert seconds to hours, minutes, and seconds
